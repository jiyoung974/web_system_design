개인정보 처리방침

Safety 365는 이용자의 개인정보를 수집하거나 저장하지 않습니다.

1. 수집하는 개인정보 항목

본 사이트는 이름, 연락처, 위치정보 등의 개인정보를 수집하지 않습니다.
단, 사용자의 현재 위치를 지도에 표시하는 기능은 브라우저의 위치정보 권한을 활용하며,
해당 정보는 서버에 저장되지 않습니다.

2. 위치정보 이용 안내

현재 위치 기반 서비스는 브라우저의 위치 접근 동의를 받은 후에만 사용됩니다.
사용자의 위치 정보는 지도 표시 용도로만 활용되며, 별도로 수집하거나 저장하지 않습니다.

3. 외부 서비스 이용

본 사이트는 Kakao 지도 API를 사용하며, Kakao 플랫폼의 위치 표시 기능이 포함될 수 있습니다.
Kakao의 위치 API 사용과 관련한 사항은 Kakao의 개인정보 처리방침을 참고해 주시기 바랍니다.

4. 개인정보 관련 문의

본 사이트는 학습 목적의 프로젝트 사이트로, 별도의 개인정보 보호 책임자가 지정되어 있지 않습니다.
문의 사항은 example@safety365.com 으로 연락 주시기 바랍니다.
